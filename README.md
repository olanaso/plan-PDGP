## Mejora del Plugin para AutoCAD – GeoTupu
### Optimización y Nuevas Funcionalidades
- **Generación de polígonos** para CAD y GIS a partir de información de coordenadas.
- Integración fluida entre entornos CAD y SIG.
- **Creación de scripts** para generación automática de planos.
- Optimización de tiempos y reducción de errores manuales.

### Beneficios
- Ahorro de tiempo en la digitalización y edición de planos.
- Mayor precisión en delimitación geográfica.
- Compatibilidad directa con visores y sistemas GIS.

**Flujo de trabajo:**
1. **Coordenadas**
2. **GeoTupu**
3. **Polígonos CAD / GIS**
4. **Planos**
